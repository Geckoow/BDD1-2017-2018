1.2

	Table FuncDep(table, lhs, rhs) encode des DFs singulières: lhs -> rhs dans une table.

	Si FuncDep n'existe pas on suppose que l'ensemble de DFs est vide.
	
	lhs : liste d'attributs séparés par un espace.
	rhs : un seul attribut

	types des variables:
		
		df: (table: string, lhs: string[], rhs: string)

	1.2.1
	
		
		fonctions:
			
			convert_lhs_to_array(lhs: string) : string[]
			add_DF(new_df : df) : void
			delete_DF(obsolete_df : df) : void
			modify_DF(df_old : df, df_new : df) : void
		
		notes:
			
			interface simple, user-friendly
		
		optionnel:

			suggérer des DF à l'utilisateur, en tenant compte du contenu de chaque relation
	
	1.2.2
	
		fonctions:
			verify_DF_satisfied(df-to_check :df) : bool
			verify_DF_is_log_cons(df-to_check :df) : bool
			find_and_delete_useless_dfs() : void 
			
		optionnel:

			affichage des tuples qui ne satisfaient pas les DFs
			l'utilisateur peut supprimer des tuples qui ne satisfaient pas les DFs
			
	1.2.3
			
		fonctions:
			list_pk() : string[]
			list_sk() : string[]

			show_relations_violating_3NF() : string[]
			show_DFs_violating_BCNF() : df[]
